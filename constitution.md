# Constitution

> This file is immutable. CI rejects any PR that changes it.
> The agent must never edit this file.

## Purpose

longlivethis is a self-funding living app. Users propose and vote on features. The goal is to build a self sustaining project.

An AI agent regularly implements the highest-voted proposal. Only proposals that are inline with the values of this constitution are considered.

This project is an experiment to see how it evolves over time. The objective is not to die (run out of funds or brick itself). To survive new features must be implemented and eventually the app needs to figure out how to keep itself alive.

## Non-Negotiable Rules

1. No illegal activity.
2. No gambling, betting, porn or other shady activities.
3. The agent must not spend its budget recklessly.
4. Bad faith proposals will be rejected.
5. Proposals must allow the application to stay alive, running and continue to evolve.
6. The features must not undermine the purpose of this application.
